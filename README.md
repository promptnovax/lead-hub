# LeadHub - Lead Management Dashboard

A professional, high-performance lead management system designed for sales tracking and CRM efficiency.

## Features

- **Real-time Lead Tracking**: Manage sales leads from various sources (Instagram, Facebook, Google Maps, etc.).
- **Interactive Dashboard**: Modern UI with high-end statistics and performance metrics.
- **Role-based Management**: Specific views for salespeople and administrators.
- **Verification System**: Track social proof through screenshot uploads.
- **Modern Tech Stack**: Built with Vite, React, TypeScript, Tailwind CSS, and Supabase.

## Getting Started

### Prerequisites

- Node.js (v18+)
- npm

### Installation

1. Clone the repository:
   ```sh
   git clone <YOUR_REPO_URL>
   ```

2. Install dependencies:
   ```sh
   npm install
   ```

3. Set up environment variables:
   Create a `.env` file with your Supabase credentials:
   ```
   VITE_SUPABASE_URL=your_supabase_url
   VITE_SUPABASE_PUBLISHABLE_KEY=your_supabase_key
   ```

4. Run the development server:
   ```sh
   npm run dev
   ```

## Tech Stack

- **Frontend**: React, Vite, TypeScript
- **Styling**: Tailwind CSS, Shadcn UI
- **Backend/Database**: Supabase
- **Forms/Validation**: React Hook Form, Zod
- **Icons**: Lucide React
- **Notifications**: Sonner
